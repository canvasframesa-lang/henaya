# 🌿 حنايـة — Henaya

متجر إلكتروني متكامل لمنتجات الجمال الطبيعي: **الحناء، السدر، كحل الإثمد، الزيوت، والأعشاب**.
هوية أنثوية بلون الحناء الأخضر · واجهة عربية RTL · بدون أي عملية بناء.

---

## ✨ المميزات

- **7 صفحات متكاملة:** الرئيسية · المنتجات · تفاصيل المنتج · السلة · الدفع · قصتنا · التواصل
- **15 منتج** بصور وأوصاف وفوائد وطريقة استخدام كاملة
- **سلة فعّالة** تحفظ في `localStorage` (تبقى بعد إعادة التحميل)
- **فلاتر بحث** بالتصنيف والسعر والترتيب
- **معرض صور للمنتج** مع صور مصغّرة قابلة للتبديل
- **عملية دفع** كاملة مع نموذج توصيل + ٤ طرق دفع + نافذة تأكيد
- **شعار SVG** بلون الحناء الأخضر + لمسة ذهبية
- **تجاوب كامل** للجوال والتابلت والديسكتوب
- **شريط مزايا** (طبيعي · توصيل سريع · دفع آمن · دعم 7 أيام)
- **خصم افتتاحي** + بانر شحن مجاني فوق ٢٠٠ ر.س
- **احتياط للصور** — إذا فشل تحميل صورة، تظهر بطاقة مصممة بألوان الهوية بدلاً من صورة مكسورة

## 🚀 التشغيل المحلي

لا يوجد build، لا يوجد npm install. فقط:

### الخيار 1: افتح الملف مباشرة
```
انقري مرتين على index.html
```

### الخيار 2: سيرفر محلي بسيط (مفضّل لتفادي قيود CORS)
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```
ثم افتحي: <http://localhost:8000>

## ✏️ التعديل والتخصيص

### إضافة/تعديل منتج
حرّري `assets/js/data.js` — كل منتج هو كائن في `HENAYA_PRODUCTS`:
```js
{
  id:'unique-id',
  name:'اسم المنتج',
  cat:'henna',          // henna | sidr | kohl | herbs | oils | tools
  price:45,
  oldPrice:55,          // اختياري — يظهر السعر القديم مشطوباً
  weight:'250 جرام',
  badge:'الأكثر مبيعاً', // اختياري
  rating:4.9, reviews:312,
  short:'وصف مختصر',
  desc:'وصف كامل',
  benefits:['فائدة 1','فائدة 2'],
  use:'طريقة الاستخدام',
  images:[url1, url2, url3]
}
```

### استبدال صور المنتجات بصور حقيقية
- ضعي صور منتجاتك في `assets/img/products/`
- بدّلي الروابط في `data.js` من `IMG.henna1` إلى مثلاً `'assets/img/products/henna-madina.jpg'`
- الأبعاد المثالية: مربع `800×800`، خلفية بيضاء أو كريمية

### تغيير ألوان الهوية
في `assets/css/styles.css` أعلى الملف، عدّلي متغيرات `:root`.

### تغيير الشعار
استبدلي `assets/img/logo.svg` بشعارك (أو حافظي على نفس الأبعاد `220×80`).

## 🔌 خطوات المستقبل (للتطوير)

عند الجاهزية للإطلاق الفعلي:

1. **بوابة دفع حقيقية:** ربط Moyasar أو Tap Payments أو HyperPay (السعودية)
2. **قاعدة بيانات:** نقل المنتجات من JS إلى Firebase / Supabase
3. **لوحة تحكم:** صفحة إدارة المنتجات والطلبات
4. **بريد الطلبات:** ربط Formspree أو EmailJS لاستقبال الطلبات على إيميل
5. **تتبع التحليلات:** إضافة Google Analytics 4 أو Plausible
6. **شهادة SSL:** تتفعّل تلقائياً مع Netlify / Vercel / Cloudflare
7. **سجل تجاري:** إضافة الرقم وشعارات الدفع في الفوتر
8. **سياسات قانونية:** صفحات الخصوصية والاسترجاع والشروط

## ⚖️ ملاحظة حول الصور

الصور الحالية من **Unsplash** (مجانية الاستخدام التجاري وفق رخصتها) كصور توضيحية احترافية.
عند الإطلاق التجاري، يُفضّل **استبدالها بصور منتجاتك الحقيقية** بالتغليف الخاص بك لزيادة المصداقية والاتساق البصري.
إذا فشل تحميل أي صورة، يعرض الموقع بطاقة بديلة بألوان الهوية تلقائياً (لا توجد صور مكسورة).
